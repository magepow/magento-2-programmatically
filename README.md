## Magento 2 programmatically

Sometime you want run a simple task but don't want create an extension or add code or custom in source magento you can use a Magento 2 programmatically run outside.

### Highlight Features
Add attribute, add attribute option, remove attribute product, remove attribute category, rename attribute, add cucstom option, add role, add schema, add user admin, apply layout category, apply layout page, apply to attribute, change type theme, flush cache, delete category, magento cli, get salable quantity, template processor update config, update product


Visit our site [Magepow.com](https://magepow.com/)

### Detail User Guide
* Copy code to root magento after go to your domain run this file exampe: domain.com/cacheflush.php
